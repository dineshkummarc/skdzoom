How to use?

Available Options

Option Description
radius Radius of the rounded box (Big Image box). Example: 150
circleOffset A number that can be used to djust distance between thubnail and rounded box (Big Image box). Example: 1.5 default value is 2
lensRadius Lens Radius. Example: 10
lensZoom true/false - Default value is true. Enable or disable lens zooming
imageBorder Default value is 8. Rounded box border width
imageBorderColor Default value is #000000. Rounded box border color
imageBorderColor Default value is #000000. Rounded box border color
imageBG Default value is #ffffff. Rounded box background color
circleBorderColor Default value is #000000. Small circle border color
circleBG Default value is #ffffff. Small circle background color
fadeSpeed Default value is 500. Fading Speed in micro seconds. Example: 500
